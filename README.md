# Random User Fetch App

A React + Vite application that fetches and displays random user data using FreeAPI.

## Features

- Fetch random user data from API
- Dynamic user profile rendering
- Fast and responsive UI
- Built with React and Vite

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Fetch API

## API Used

FreeAPI.app

## Installation

Clone the repository:

```bash
git clone <your-repo-url>
```

Go to project folder:

```bash
cd your-project-name
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```


## Learning Outcome

Built this project to practice API fetching, async JavaScript, state management, and React fundamentals.
